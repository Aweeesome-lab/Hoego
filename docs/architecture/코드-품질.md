# 코드 품질

## Formatting

```yaml
Rust:
  - rustfmt (자동)
  - 실행: cargo fmt

TypeScript:
  - Prettier (자동)
  - 들여쓰기: 2칸
  - 세미콜론: 사용
  - 따옴표: 작은따옴표
```

## Linting

```yaml
Rust:
  - clippy
  - 실행: cargo clippy -- -D warnings

TypeScript:
  - ESLint + Prettier
  - 실행: npm run lint
```

## 테스팅

```yaml
Rust:
  - 단위 테스트: #[cfg(test)]
  - 실행: cargo test

TypeScript:
  - Vitest + Testing Library
  - 실행: npm run test
  - Coverage: npm run test:coverage
```
