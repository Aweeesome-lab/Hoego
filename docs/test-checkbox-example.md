## 테스트 포인트

확인해야 할 사항:

1. ✅ 체크박스 앞에 불릿이 표시되지 않음
2. ✅ 체크박스가 클릭 가능함
3. ✅ 클릭 시 상태가 토글됨
4. ✅ 중첩 체크박스도 정상 작동
5. ✅ 일반 리스트와 체크박스가 구분됨
